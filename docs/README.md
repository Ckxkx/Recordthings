## 📋 制作规范

### 🎨 设计规范
- **主色调**：紫色渐变 (#667eea → #764ba2)
- **辅助色**：绿色(#4cd964)、橙色(#ff9500)、红色(#ff3b30)
- **字体**：系统默认字体
- **圆角**：统一使用20rpx圆角

### 📱 尺寸规范
- **手机截图**：375x812px (iPhone X尺寸)
- **GIF动图**：375x812px，文件大小 < 5MB
- **展示图片**：1200x800px
- **Logo**：512x512px

### 🎬 动画规范
- **帧率**：15-20 FPS
- **时长**：5-15秒
- **循环**：无缝循环
- **压缩**：保证清晰度的前提下尽量压缩文件大小

## 🛠️ 制作工具推荐

### 📸 截图工具
- **macOS**：Command + Shift + 4
- **Windows**：Snipping Tool / Snagit
- **在线工具**：Figma、Canva

### 🎬 GIF制作
- **LICEcap**：轻量级屏幕录制
- **GIPHY Capture**：Mac专用
- **ScreenToGif**：Windows专用
- **在线工具**：ezgif.com

### 🎥 视频制作
- **OBS Studio**：免费屏幕录制
- **QuickTime**：Mac内置录制
- **Camtasia**：专业视频编辑
