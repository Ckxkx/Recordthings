# ⚡ 快速开始

> **30秒运行RecordThings，立即体验智能库存管理！**

## 🚀 一键运行

### 方法一：HBuilderX（推荐）

```bash
# 1️⃣ 克隆项目
git clone https://github.com/your-username/RecordThings.git

# 2️⃣ 用HBuilderX打开项目文件夹

# 3️⃣ 点击运行按钮
运行 → 运行到小程序模拟器 → 微信开发者工具
```

### 方法二：命令行

```bash
# 克隆并运行
git clone https://github.com/your-username/RecordThings.git
cd RecordThings
npm run dev:mp-weixin
```

## 🎯 首次体验

项目运行后会自动加载示例数据，你可以：

1. **📊 查看总览**：在首页查看库存统计
2. **➕ 添加物品**：点击右下角"+"按钮
3. **📈 查看统计**：切换到"更多"页面，点击"数据统计"
4. **🔍 搜索物品**：使用搜索功能查找物品

- 📊 **数据可视化** - 了解库存结构  
- 🔍 **实时搜索** - 快速找到物品
