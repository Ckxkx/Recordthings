# 底部导航栏图标说明

请在此目录下放置以下图标文件：

## 需要的图标文件

1. **overview.png** - 总览图标（未选中状态）
   - 建议尺寸：81px × 81px
   - 建议使用房子图标

2. **overview-active.png** - 总览图标（选中状态，蓝色）
   - 建议尺寸：81px × 81px
   - 建议使用房子图标，填充蓝色 #007AFF

3. **inventory.png** - 库存图标（未选中状态）
   - 建议尺寸：81px × 81px
   - 建议使用文件夹/盒子图标

4. **inventory-active.png** - 库存图标（选中状态，蓝色）
   - 建议尺寸：81px × 81px
   - 建议使用文件夹/盒子图标，填充蓝色 #007AFF

5. **more.png** - 更多图标（未选中状态）
   - 建议尺寸：81px × 81px
   - 建议使用人物图标

6. **more-active.png** - 更多图标（选中状态，蓝色）
   - 建议尺寸：81px × 81px
   - 建议使用人物图标，填充蓝色 #007AFF

## 图标资源

您可以从以下网站获取免费图标：
- [Iconfont](https://www.iconfont.cn/)
- [Iconify](https://iconify.design/)
- [Material Icons](https://fonts.google.com/icons)

或者使用简单的占位图标也可以正常运行。
